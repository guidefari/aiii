# aiii

To install dependencies:

```bash
bun install
```

To run:

1. create an [openai API key](https://platform.openai.com/api-keys)
2. create a .env file
```
   OPENAI_API_KEY=''
```

```bash
bun dev
```

This project was created using `bun init` in bun v1.0.35. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
